Detailed Description

template<typename T, typename C, typename D>
class morpheus::memory::detail::indirect_value_base< T, C, D >

We must specialise to handle the case where copier and deleter are the same object and have no size. In this case this we have [[no_unique_address]] pointing to two objects of the same type which is not allowed by the standard. To support this we specialises away the unnecessary duplication of the object to avoid unnecessary storage and in-doing so resolve the any issues around use of [[no_unique_address]]: https://en.cppreference.com/w/cpp/language/attributes/no_unique_address
