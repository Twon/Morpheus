Detailed Description

template<typename T>
class morpheus::concurrency::Generator< T >

Minimal generator type based on coroutines which models std::ranges::input_range. Largely inspired as a temporary solution to bridge missing support for this unil the inclusion of std::generator: https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2022/p2502r2.pdf
