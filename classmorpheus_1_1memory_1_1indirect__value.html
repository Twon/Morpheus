class morpheus::memory::indirect_value< T, Copier, Deleter >

Implements P1950R2, a free-store-allocated value type for C++

Template Parameters
TThe underlying value type.
CopierThe copier functor to customise how the underlying value is copied.
DeleterThe deleter functor to customise how the underlying value is deleted.
