Detailed Description

The compiler group of macros allow for compile time detection of the current compiler and version.

The compiler group of macros allow for compile time detection of the sanitizers in the build.

Note
https://clang.llvm.org/docs/AddressSanitizer.html#conditional-compilation-with-has-feature-address-sanitizer https://docs.microsoft.com/en-us/cpp/sanitizers/asan-building?view=msvc-170
